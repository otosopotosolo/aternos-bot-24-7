## Impact

**CRITICAL:** Arbitrary origin + credentials = any website can:
1. Steal user sessions/cookies
2. Access sensitive API data
3. Perform actions as logged-in user

**HIGH:** Arbitrary origin reflected with credentials:
1. Cross-origin data theft possible
2. Session hijacking if cookies are used
